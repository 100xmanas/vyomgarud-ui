# Overview

Frontend landing page for VyomGarud. Fast, modern, and responsive UI built with React, Tailwind CSS, and Framer Motion.

# Tech Stack

- **Frontend:** React.js
- **Styling:** Tailwind CSS (Responsive design, utility-first styling)  
- **Animations:** Framer Motion (Smooth hover and page transitions)  
- **Icons:** Lucide-react
